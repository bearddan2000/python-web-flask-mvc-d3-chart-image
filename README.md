# python-web-flask-mvc-d3-chart-image

## Description
A POC for showing d3
javascript framework
donut chart in a flask mvc.

## Tech stack
- d3js
- python
  - flask
  - pandas

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`
- Available http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
